<Step>
## 1. Initialize shadcn/ui

Run the `shadcn/ui` init command in your project root. This will set up `components.json` and configure necessary styles and utilities.

<CodeTabs lang="bash" codes={{
    bash: `pnpm dlx shadcn@latest init`,
  }} />

You will be prompted to select a base color:

```text
pnpm dlx shadcn@latest init
✔ Preflight checks.
✔ Verifying framework. Found Next.js.
✔ Validating Tailwind CSS config. Found v4.
✔ Validating import alias.
? Which color would you like to use as the base color? › - Use arrow-keys. Return to submit.
❯   Neutral
    Gray
    Zinc
    Stone
    Slate
```

This will:
- Create or update `components.json` to manage component dependencies.
- Create `lib/utils.ts` (or `.js`) with helper functions like `cn`.
- Update your global CSS file (`globals.css`) with Tailwind directives and CSS variables for themes.
- Update `tailwind.config.ts` (or `.js`/`.mjs`) to include shadcn styles.
</Step>

<Step>
## 2. Add Components

Add individual components as needed using the `add` command. You can add multiple components at once.

<CodeTabs lang="bash" codes={{
    'Add Components': `pnpm dlx shadcn@latest add button card input`,
    'Add All Components': `pnpm dlx shadcn@latest add --all`,
  }} />

This command downloads the source code for the requested components directly into your project, typically under `components/ui/`. You can then modify them as needed.
</Step>

<Step>
## 3. (Optional) Dark Mode Setup with next-themes

Shadcn/ui works well with `next-themes` for dark mode support.

### 3a. Install `next-themes`

<CodeTabs lang="bash" codes={{
    bash: `pnpm add next-themes`,
  }} />

### 3b. Create Theme Provider

Create a theme provider component. This wraps your application and provides the theme context.

<CodeTabs lang="tsx" codes={{
    'components/theme-provider.tsx': `'use client'

import * as React from "react"
import { ThemeProvider as NextThemesProvider } from "next-themes"
import { type ThemeProviderProps } from "next-themes"

export function ThemeProvider({ children, ...props }: ThemeProviderProps) {
  return <NextThemesProvider {...props}>{children}</NextThemesProvider>
}`,
  }} />


*(Place this file in `components/theme-provider.tsx` or `src/components/theme-provider.tsx`)*

### 3c. Update Root Layout

Wrap your root layout (`app/layout.tsx` or `src/app/layout.tsx`) with the `ThemeProvider`.

<CodeTabs lang="tsx" codes={{
    'app/layout.tsx': `import type { Metadata } from "next";
import { Inter } from "next/font/google";
import "./globals.css";
import { ThemeProvider } from "@/components/theme-provider"; // Adjust path if needed

const inter = Inter({ subsets: ["latin"] });

export const metadata: Metadata = {
  title: "Create Next App",
  description: "Generated by create next app",
};

export default function RootLayout({
  children,
}: Readonly<{
  children: React.ReactNode;
}>) {
  return (
    <html lang="en" suppressHydrationWarning>
      <body className={inter.className}>
        <ThemeProvider
          attribute="class"
          defaultTheme="system"
          enableSystem
          disableTransitionOnChange
        >
          {children}
        </ThemeProvider>
      </body>
    </html>
  );
}`,
  }} />


### 3d. Create Mode Toggle Button

Create a component to allow users to switch between light and dark modes. This component typically uses `lucide-react` icons. Ensure `lucide-react` is installed (`pnpm add lucide-react`).

<CodeTabs lang="tsx" codes={{
    'components/mode-toggle.tsx': `'use client'

import * as React from "react"
import { Moon, Sun } from "lucide-react"
import { useTheme } from "next-themes"

import { Button } from "@/components/ui/button" // Adjust path if needed
import {
  DropdownMenu,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuTrigger,
} from "@/components/ui/dropdown-menu" // Adjust path if needed

export function ModeToggle() {
  const { setTheme } = useTheme()

  return (
    <DropdownMenu>
      <DropdownMenuTrigger asChild>
        <Button variant="outline" size="icon">
          <Sun className="h-[1.2rem] w-[1.2rem] rotate-0 scale-100 transition-all dark:-rotate-90 dark:scale-0" />
          <Moon className="absolute h-[1.2rem] w-[1.2rem] rotate-90 scale-0 transition-all dark:rotate-0 dark:scale-100" />
          <span className="sr-only">Toggle theme</span>
        </Button>
      </DropdownMenuTrigger>
      <DropdownMenuContent align="end">
        <DropdownMenuItem onClick={() => setTheme("light")}>
          Light
        </DropdownMenuItem>
        <DropdownMenuItem onClick={() => setTheme("dark")}>
          Dark
        </DropdownMenuItem>
        <DropdownMenuItem onClick={() => setTheme("system")}>
          System
        </DropdownMenuItem>
      </DropdownMenuContent>
    </DropdownMenu>
  )
}`,
  }} />


*(Place this file in `components/mode-toggle.tsx` or `src/components/mode-toggle.tsx`)*

You can now import and use `<ModeToggle />` anywhere in your application (e.g., in your site header).
</Step>

<Step>
## 4. Usage Example

Now you can import and use the installed shadcn/ui components in your pages or components.

<CodeTabs lang="tsx" codes={{
    'app/page.tsx': `import { Button } from "@/components/ui/button" // Adjust path if needed
import { ModeToggle } from "@/components/mode-toggle"; // Adjust path if needed

export default function Home() {
  return (
    <main className="flex min-h-screen flex-col items-center p-24">
      <div className="absolute top-4 right-4">
        <ModeToggle />
      </div>
      <h1 className="text-4xl font-bold mb-4">Welcome to shadcn/ui</h1>
      <Button>Click Me</Button>
    </main>
  );
}`,
  }} />
</Step>
